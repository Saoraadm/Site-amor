# Site Amor

Projeto estático com páginas românticas para guardar memórias do casal.

## Páginas

- `index.html`: página inicial do site.
- `viagens.html`: galeria dos passeios e viagens.
- `fotos.html`: mural de fotos com legendas engraçadas.
- `aniversario.html`: contador de tempo juntos.

## Como executar localmente

Como é um site estático, inicie um servidor local na raiz do projeto:

```bash
python3 -m http.server 4173
```

Depois acesse no navegador:

- `http://localhost:4173/index.html`
- `http://localhost:4173/fotos.html`
